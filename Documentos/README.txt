Sobre o projeto Minhas Vacinas;

Linguagem:PHP;
Framework: Laravel;
Jquery;
Repositório: github
BD PHPmyadmin;

O projeto tem como objetivo o registro de usuarios sendo eles classificados por funções onde possuem 
funções especificas, uso de ACL em laravel, também foi usado migration em Laravel para criação
de tabelas no BD, todo o projeto foi commitado em um repositório no Git , disponivel em: https://github.com/fabiogresko/Carteira-de-vacinacao.git,
as funções disponiveis no sistemas são registro de usuarios, registro de Vacinas, atribuição de função aos usuarios, alterar e remover usuarios e
 vacinas,Visualização de usuarios, carteira de vacinação, funções e permissões no sistema minhas vacinas.
